## Run Locally

Clone the project

```bash
  git clone https://github.com/ayusshrathore/react-native-wallet
```

Go to the project directory

```bash
  cd react-native-wallet
```

Install dependencies

```bash
  npm install
```

Start the expo server

```bash
  npx expo start
```
